Detailed Description

Transport Abstraction Layer (TAL) API.

The set of functions in this module provide an abstraction layer between the radio (companion) module and the underlying hardware.

ADI_BLE_TRANSPORT_RESULT adi_tal_Read ( void *  pBuf,
const uint32_t  nBufSize,
uint8_t *  nBytes 
)

Transport Layer Read.

Reads nBufSize bytes tp pBuf from the device

Parameters
[in]pBuf: Pointer to the destination buffer
[in]nBufSize: Number of valid bytes in pBuf to read
[out]nBytes: Number of valid bytes read into pBuf. May be less than nBufSize but not more.
Returns
ADI_BLE_TRANSPORT_RESULT
  • ADI_BLE_TRANSPORT_SUCCESS No error occurred
  • ADI_BLE_TRANSPORT_RDY_TIMEOUT Timed out waiting for RDY signal
  • ADI_BLE_TRANSPORT_INVALID_DATA_SIZE Data either overflowed or underflowed buffer so no trasnaction occured
  • ADI_BLE_TRANSPORT_FAILED_GPIO GPIO Failed
  • ADI_BLE_TRANSPORT_FAILED_TRANSACTION SPI transaction failed
