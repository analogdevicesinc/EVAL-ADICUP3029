Detailed Description

Structure that contains data returned during an inquiry procedure. While the remote address, RSSI, and mode are always present, the remaining advertising data is variable, which is why the buffer structure is used. The application can parse and filter the advertising data using the respective APIs to glean more information about the meaning of the data.
